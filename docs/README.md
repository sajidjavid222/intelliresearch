# Screenshots

Drop screenshots of the running app here, then uncomment the image lines in the
root [`README.md`](../README.md) so they show on the GitHub repo page.

Suggested shots (from the live demo or your local app):

| File name | What to capture |
|-----------|-----------------|
| `search.png` | The search page with results (e.g. search “diffusion models”). |
| `graph.png` | The **Graph** page — the citation network. |
| `dashboard.png` | The dashboard with collections / saved items. |

**Easiest way:** open the app, take a screenshot (`Cmd+Shift+4` on macOS),
save it here with the name above, then in the GitHub web UI uncomment the
`![...](docs/…)` lines in the README. They'll render instantly.
